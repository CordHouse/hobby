* [최초 실행시]

1. 처음 프로그램 사용시 빨간 버튼을 클릭해 모두 초기화를 진행해 줍니다.

2. 시간값 메모장을 키고 첫번째 빈칸과 두번째 빈칸에 각각 시간과 분을 작성 합니다.


* [시간 기록이 있었던 경우]
  *주의 : 처음 배포받고 자신의 기록을 유지해 나갈때만 사용하세요!

1. 디스코드 접속기록 보고서에 들어가서 맨 아래부터
(접속날짜, 접속시간, 종료시간, 유지시간, "====") 라인까지만 복사 해준다.

2. 골드문 방탄 계산기 폴더에 시간값 메모장에 아랫순서로 작성해준다.
총누적시간 ( 초기값 빈칸 )
총누적분 ( 초기값 빈칸 )
누적시간 ( 초기값 0으로 설정되어있음 )
누적분 ( 초기값 0으로 설정되어있음 )

3. 다 되었다면 프로그램에 있는 가져오기 버튼을 눌러주고 접속버튼을 클릭해준다.


* [방탄 총 누적 수 설정]

1. 구분 값 설정 후 금액과 총 누적개수를 입력해 줍니다. ( 아무거나 상관없음 )

2. 방탄 버튼 클릭 -> 누적 저장하기 클릭 -> 누적 값 클릭 -> 총 누적값 클릭
으로 값이 오른쪽 위에가 바뀌었는지 확인합니다.

3. 바뀌었다면 방탄 갱신 버튼을 클릭합니다.
( 버튼순서 : 방탄 갱신 - > 방탄 -> 누적저장 -> 누적값 -> 총누적값 )


* [방탄 평시 판매]

1. 구매자의 구분 판매금 판매개수를 입력해 줍니다.
  *주의 : 구분이 바뀌면 해당 개수에 변경버튼도 꼭 눌러주세요

2. ( 버튼순서 : 누적값 -> 총누적값 -> 방탄 -> 누적저장 )순으로 클릭 해줍니다.


* [신호탄 판매]

1. 구분과 개수만 입력하고 총신인지 총알인지 선택해줍니다.

2. 신호탄 버튼을 클릭해 줍니다.


* [검은돈]

1. 거래자의 구분을 선택해줍니다.

2. 신청금액란에 금액을 입력해줍니다.

3. 검은돈 버튼을 클릭해줍니다.


* [브로커]

1. 적출장기인지 인공장기인지 선택해 줍니다.

2. 장기를 선택해 줍니다.

3. 장기버튼을 클릭해 줍니다.
